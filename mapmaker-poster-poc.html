Poster Tiler POC

Pan / zoom the map to frame the area you want as a wall poster, set the options, then generate a tiled PDF.

Move the map to see the estimate.

Basemap: defaults to OpenStreetMap (no key needed) so this runs immediately. For crisp enlargement at poster scale, set CONFIG.apiKey and switch CONFIG.basemap to a vector basemap (e.g. "topo-vector") near the top of the script — vector re-renders sharp at any size, raster does not.
